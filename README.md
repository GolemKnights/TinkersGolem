# Tinkers' Golem
## Introduction
This is an addon of [ModularGolem](https://github.com/Minecraft-LightLand/ModularGolems) and [Tinkers' Construct](https://github.com/SlimeKnights/TinkersConstruct/).
It does a lot of things:
- Tinkers' armors and weapons of metal golem;
- Tinkers' armors of dog golem;
- Golem made of slime metals;(overslime acts as absorption!)
- Slime golem;
- Golem materials from tinkers' addons, such as tinkers' thinking.
